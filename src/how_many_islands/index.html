Description

Syntax with textarea

c11 c12 ... c1m

c21 c22 ... c2m

... ... ... ...

cn1 cn2 ... cnm

cij should be 0 or 1.

API

  • inputFromTextarea()
  • dfsGen(map)
  • bfsGen(map)
  • ufGen(map)
  • update(map, td_width, td_height, colfunc)

  • formatForGen(string)

    string should be the syntax written above.

    sample (JavaScript syntax):

    const str = "\

    1 1 1 0\n\

    1 0 0 0\n\

    1 0 1 0\n\

    1 0 1 0\n"
